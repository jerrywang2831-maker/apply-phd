# 神经科学（基础与临床）

## 方向概述

神经科学研究神经系统的结构、功能、发育和病理。国内此方向在**北京大学**（麦戈文研究所）、**复旦大学**（脑科学研究院）、**中科院神经所**（上海脑智卓越中心）拥有最强基础研究力量。

## 子方向

- **系统神经科学**：感觉/运动/认知神经回路 — 北大(方方)、中科院神经所、浙大
- **细胞与分子神经科学**：神经递质、突触传递 — 北大(李毓龙)
- **神经发育与可塑性**：轴突导向、神经再生 — 中科院神经所(蒲慕明)
- **神经退行性疾病**：AD/PD模型 — 清华、复旦、浙大
- **计算神经科学**：神经信息编码、神经网络模型 — 北大(吴思)、浙大(江海腾)
- **认知神经科学**：注意/记忆/语言 — 北大(方方、纳家勇治)

## 针对你的申请建议

神经科学方向与你的背景匹配度分析：

- **计算神经科学**：⭐⭐⭐⭐ — 数学/编程能力强是优势，与你的MATLAB/Python背景匹配
- **神经退行性疾病模型**：⭐⭐⭐⭐ — 脑类器官(AD模型)经验直接对口
- **系统神经科学**：⭐⭐⭐ — 需补充更多体内电生理实验经验
- **分子神经科学**：⭐⭐ — 需要更多分子生物学和生化实验技能

建议策略：以你的iPSC/脑类器官研究经验为切入点，申请偏向**神经退行性疾病模型**或**类器官技术平台**方向的课题组。北京大学李毓龙（神经递质探针）和复旦大学的脑类器官相关课题组值得重点关注。
