# 生物医学工程 (BME)

## 方向概述

生物医学工程将工程学原理应用于医学和生命科学领域。国内BME方向在**上海交通大学**、**华中科技大学**、**东南大学**拥有国家一级重点学科，清华大学、浙江大学也在快速发展。

## 子方向

- **医疗器械与系统**：超声治疗、微创诊疗 — 清华(廖洪恩)、上交
- **生物材料与组织工程**：纳米材料、再生医学 — 川大(张兴栋院士)、清华(杜亚楠)
- **生物力学**：骨骼/心血管力学 — 北航
- **康复工程**：运动功能障碍康复 — 中科院(侯增广)
- **生物传感器与微流控**：生物芯片、纳米传感器 — 清华(程京院士、李舟)
- **神经工程（BME视角）**：神经假体、脑机接口硬件 — 清华(洪波)、浙大

## 针对你的申请建议

BME方向对你而言是**最宽口径**的选择：

- **生物医疗仪器方向**最匹配你的背景（超声RA经验 + EMG硬件开发）
- **神经工程**在BME语境下更偏硬件/设备，适合你的工程能力
- **生物材料/组织工程**方向需要更多化学/材料学背景，匹配度较低

建议申请BME项目时，在你的研究计划中重点突出**工程实践能力**（电路设计、系统集成、原型开发），这是许多纯生物/医学背景申请人不具备的差异化优势。
