# 研究方向总览

## 四大研究方向

### 1. [神经工程 / 脑机接口 (BCI)](/research-directions/bci)

脑机接口技术涉及神经信号的采集、解码与反馈，是神经科学与工程技术的交叉前沿。

**子方向**：
- 侵入式BCI（微电极阵列、ECoG）
- 非侵入式BCI（EEG、fNIRS）
- 神经信号处理与解码算法
- 神经调控（DBS、TMS、tACS）
- 闭环神经反馈系统
- 神经假体与康复工程

**相关院校**：清华、浙大、天大、中科院自动化所、上交

### 2. [生物医学工程 (BME)](/research-directions/bme)

生物医学工程将工程学原理应用于医学和生物领域，涵盖从分子到器官尺度的技术开发。

**子方向**：
- 医疗器械与系统
- 生物材料与组织工程
- 生物力学
- 康复工程
- 生物传感器与微流控
- 医学影像设备

**相关院校**：上交、华科、东南、北航、川大

### 3. [神经科学（基础与临床）](/research-directions/neuroscience)

研究神经系统的结构、功能、发育和病理，从分子层面到系统层面理解大脑。

**子方向**：
- 系统神经科学（感觉、运动、认知回路）
- 细胞与分子神经科学
- 神经发育与可塑性
- 神经退行性疾病（阿尔茨海默病、帕金森病）
- 神经精神疾病
- 计算神经科学

**相关院校**：北大、复旦、中科院神经所、浙大

### 4. [医学物理 / 影像](/research-directions/medical-physics)

运用物理学原理和方法解决医学问题，涵盖成像、治疗和测量技术。

**子方向**：
- 磁共振成像 (MRI)
- 超声成像与治疗
- 光学成像（双光子/共聚焦/近红外）
- 核医学与PET
- 放射治疗物理
- 医学图像处理与分析

**相关院校**：华科、东南、中山、中科大
