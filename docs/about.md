# 关于本站

## 网站目的

本站为个人整理的 **2026年秋季入学中国博士申请一站式信息平台**，旨在系统性地汇总中国顶尖高校在神经工程、生物医学工程及相关方向的博士招生信息。

**建设目标**：
- 一站式查阅目标院校的导师信息、研究方向、申请流程
- 按研究方向交叉索引，快速定位匹配的导师
- 跟踪2026年秋季入学的关键申请时间节点
- 所有信息标注来源，链接可追溯、可验证

## 数据来源声明

本站所有信息来源于以下渠道（按优先级排序）：

1. 各院校研究生院/学院官网公告
2. 导师个人主页（.edu.cn 域名）
3. 中国知网 (CNKI)、PubMed、Google Scholar
4. 官方微信公众号/通知公告

所有外部链接均标注访问日期。信息具有一定时效性，请以各校官方最新公告为准。
