# 申请材料准备

## 通用材料清单

| 序号 | 材料名称 | 要求 | 准备时间 |
|------|----------|------|----------|
| 1 | 报考登记表 | 网上报名系统自动生成，打印签字 | 报名时 |
| 2 | 个人陈述 | 1000-3000字，阐述学术背景和研究动机 | 提前4周 |
| 3 | 研究计划书 | 3000-5000字，含文献综述、方法设计 | 提前6-8周 |
| 4 | 个人简历 (CV) | 1-2页，中英文各一份 | 持续更新 |
| 5 | 硕士成绩单 | 官方版本，加盖公章 | 提前4周申请 |
| 6 | 本科学位证+成绩单 | 官方版本 | 提前4周申请 |
| 7 | 专家推荐信 | 2-3封，副教授及以上职称 | 提前6周联系 |
| 8 | 英语水平证明 | IELTS 6.5+ / TOEFL 90+ / CET-6 | 提前备考 |
| 9 | 代表性成果 | 论文、专利、获奖、项目报告 | — |
| 10 | 身份证/护照复印件 | — | — |

## 个人陈述 (Personal Statement)

### 写作结构

```
1. 开头（10%）— 一句话概括你是谁，为什么申请
2. 学术背景（20%）— 本硕学习经历，重点突出相关课程和研究
3. 研究经历（40%）— 详细描述科研项目，强调个人贡献和技能
4. 研究兴趣与动机（20%）— 为什么对这个方向感兴趣，为什么选择这个导师
5. 未来规划（10%）— 短期和长期职业目标
```

### 写作建议

- **突出海外留学背景的优势**：国际视野、中英双语能力、跨文化科研经验
- **量化成果**：如可能，用数据说明（GPA排名、论文影响因子、项目规模等）
- **个性化**：针对每个申请学校和导师定制，不要使用通用模板
- **展示科研思维**：不仅要写做了什么，更要写为什么这样做、学到了什么

### 针对你的经历的关键卖点

1. **帝国理工神经技术硕士** — 世界Top 10名校，专业对口
2. **脑类器官iPSC研究** — 前沿技术，结合工程与生物
3. **独立完成硬件+软件原型开发** — 工程实践能力
4. **UCL医用物理本科** — 扎实的物理和数学基础
5. **中英双语科研能力** — 文献阅读、论文写作无障碍

## 研究计划书 (Research Proposal)

### 写作结构

```
1. 标题 — 简洁明确，反映研究核心问题
2. 摘要（200-300字）— 研究问题、方法、意义
3. 研究背景与文献综述（约1500字）
   - 领域现状
   - 存在的问题/知识缺口
   - 本研究的切入点
4. 研究目标与科学问题（约500字）
   - 总体目标
   - 具体研究问题/假设
5. 研究方法（约1500字）
   - 实验设计/技术路线
   - 数据采集与分析方案
   - 可行性分析
6. 预期成果与创新点（约500字）
7. 时间规划（约300字）
8. 参考文献
```

### 写作建议

- 尽量与目标导师的研究方向衔接
- 展示对该领域的深入理解
- 方法论要具体、可行
- 引用该导师的论文（展示你做过功课）
- 可以基于你的硕士项目延伸

## 推荐信 (Recommendation Letters)

### 推荐人选择

| 优先级 | 推荐人 | 理由 |
|--------|--------|------|
| 1 | 硕士导师/项目导师 | 最了解你近期科研能力的人 |
| 2 | 本科导师 | 可从长期成长角度评价 |
| 3 | 课程教师 | 如果某门课成绩突出并有深入交流 |
| 4 | RA的PI/项目负责人 | 证明实际工作能力 |

### 如何联系推荐人

**邮件要点**：
- 礼貌问候，简要回顾你们的关系
- 说明你在申请博士，以及申请的方向和学校
- 提供你的简历、成绩单和研究计划供参考
- 给出明确的截止日期
- 附上推荐信模板或提交链接（如有）
- 至少提前4周联系

### 推荐信应包含的内容

- 与申请人的关系及认识时间
- 对申请人学术能力的评价（独立科研、批判性思维、实验技能）
- 具体事例支撑（如某个项目中的表现）
- 与其他学生的比较
- 对申请人读博的推荐程度

## 代表性成果

可以提交的成果类型：

- 已发表/在投的学术论文
- 本科/硕士毕业论文
- 项目技术报告
- 发明专利（授权或申请）
- 学术会议报告/海报
- 竞赛获奖证书
- GRE/GMAT成绩（如有）

## 材料提交注意事项

- **电子版格式**：通常要求PDF格式，注意文件命名规范（如：姓名-材料类型.pdf）
- **纸质版邮寄**：使用顺丰/EMS等可追踪的快递，保留快递单号
- **文件大小**：注意各校系统对上传文件的大小限制
- **彩色扫描**：学位证、成绩单等原件需彩色扫描
- **翻译件**：非中文材料需要翻译并公证
