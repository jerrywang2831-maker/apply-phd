## 联系时间建议

| 月份 | 适宜程度 | 说明 |
|------|----------|------|
| 1月-2月 | ⭐⭐ | 寒假期间，导师可能不常查邮件 |
| 3月-6月 | ⭐⭐⭐⭐ | 学期中，导师正常办公，先于正式申请季布局 |
| **6月-8月** | ⭐⭐⭐⭐⭐ | **最佳时间**，招生季即将开始（清华7月可能启动），导师开始考虑新生 |
| 9月-10月 | ⭐⭐⭐⭐ | 正式申请季前，导师收到大量邮件，回复率可能降低 |
| 11月-12月 | ⭐⭐⭐ | 申请季中后期 |

> **2026年6月-8月**是联系2027年秋季入学导师的最佳窗口期。清华可能在2026年7月启动第一批报名，建议**6月底前**完成首轮邮件发送。

## 面试/沟通准备清单

当收到导师正面回复并准备视频/线上面试时：

- [ ] 阅读导师近3年主要论文（至少5篇）
- [ ] 准备3-5分钟自我介绍（学术背景 + 研究经历 + 兴趣动机）
- [ ] 准备1-2个关于课题组研究的具体问题
- [ ] 准备研究计划的大致方向（无需最终定稿，但要有清晰思路）
- [ ] 测试视频通话设备（摄像头、麦克风、网络）
- [ ] 准备英文版本（部分导师会测试英语水平）
