## 研究方向

- **神经工程 / 脑机接口** — 脑信号采集与解码、神经调控、闭环反馈系统
- **生物医学工程** — 医疗器械、生物材料、组织工程、康复工程
- **神经科学（基础与临床）** — 神经退行性疾病、系统神经科学、认知神经科学
- **医学物理 / 影像** — 超声成像、MRI、光学成像、放射治疗物理
