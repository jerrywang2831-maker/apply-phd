# 浙江大学

> **最后更新**: 2026年5月25日

## 学校概况

浙江大学在脑机接口领域拥有国内最强的研究力量之一，依托**脑机智能全国重点实验室**和**求是高等研究院**，在植入式BCI、脑机混合智能、神经调控等方向处于国际前沿。

**官方链接**:
- 学校官网: [https://www.zju.edu.cn](https://www.zju.edu.cn)
- 研究生院: [https://grs.zju.edu.cn](https://grs.zju.edu.cn)
- 脑机智能全国重点实验室: [相关页面]
- 求是高等研究院: [相关页面]

## 相关学院

浙大BCI方向的导师分布在多个学院：
- **计算机科学与技术学院** — 潘纲、郑能干等（脑机接口算法/系统）
- **生物医学工程与仪器科学学院** — 张韶岷等（植入式BCI设备）
- **集成电路学院** — 赵博等（脑机接口芯片硬件）
- **医学院** — 神经科学与临床转化

- 材料审核：学术背景、科研成果、研究计划

- 面试：研究计划汇报 + 专家提问（含英语）

## 针对你的申请建议

1. **潘纲教授是最佳匹配目标之一**：作为脑机智能全国重点实验室主任，领导国内最大的BCI研究平台之一，你的帝国理工神经技术背景高度匹配
2. **多学院选择策略**：可根据个人偏好选择侧重算法的计算机学院或侧重硬件的生仪学院/集成电路学院
3. **提前联系导师尤为重要**：浙大BCI方向竞争激烈，建议2025年7-8月发送联系邮件
4. **脑机接口芯片方向**：赵博教授侧重芯片硬件，你的电路设计经验（EMG项目）可作切入点
5. **张韶岷研究员**的植入式BCI方向与你的脑类器官/神经技术背景可能有交叉点
