## 学校概况
北京大学在神经科学领域拥有全国最强的学科实力，汇聚了**方方**（视觉认知）、**李毓龙**（GRAB神经递质探针）、**吴思**（计算神经科学）、**饶毅**（神经生物学）等国际知名学者。

### 3. 前沿交叉学科研究院

### 4. IDG麦戈文脑科学研究所

## 针对你的申请建议

1. **最适合的方向是计算神经科学/神经工程交叉**：吴思教授的计算神经科学方向可能与你的BME神经技术背景有交叉
2. **李毓龙实验室值得关注**：神经递质探针工具与你的脑类器官研究（钙成像）有技术上的衔接，GRAB探针也可用于脑类器官
3. **北大偏好学术型候选人**：相比清华的工程导向，北大更偏重基础科学研究，研究计划需突出科学问题深度
4. **麦戈文研究所是核心平台**：汇聚了校内多个学院的PI，适合跨方向申请
5. **竞争非常激烈**：北大神经科学类博士名额有限，建议提前联系导师
