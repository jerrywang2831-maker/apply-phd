# 复旦大学

> **最后更新**: 2026年5月25日

## 学校概况

复旦大学在神经科学领域拥有**脑科学研究院**和**类脑智能科学与技术研究院**两大平台。类脑院以AI+脑科学交叉为特色，脑科学转化研究院以小胶质细胞等基础方向见长。与华山医院、中山医院等顶级医院有紧密合作。

**官方链接**:
- 学校官网: [https://www.fudan.edu.cn](https://www.fudan.edu.cn)
- 研究生院: [https://gs.fudan.edu.cn](https://gs.fudan.edu.cn)
- 类脑智能研究院: [https://istbi.fudan.edu.cn](https://istbi.fudan.edu.cn)
- 脑科学研究院: [https://iobs.fudan.edu.cn](https://iobs.fudan.edu.cn)
- 脑科学转化研究院: [https://itbr.fudan.edu.cn](https://itbr.fudan.edu.cn)

## 重点课题组与导师

### —— 类脑智能 / 计算神经科学方向 ——

#### 纪鹏 — 研究员、博导

- **学院**: 类脑智能科学与技术研究院
- **个人主页**: [https://faculty.fudan.edu.cn/pengji](https://faculty.fudan.edu.cn/pengji)

**研究方向**:
- 复杂系统与神经元网络建模

**代表性论文**: *Nature Physics*, *Nature Communications*, *Physics Reports*, *PRL*

#### 尤佳 — 青年研究员、博导

- **学院**: 类脑智能科学与技术研究院
- **荣誉**: 2025年"上海科技青年35人引领计划"（U35）、国家级青年人才

#### Gunter Schumann (冈特·舒曼) — 特聘教授、博导

- **学院**: 类脑智能科学与技术研究院（群体神经科学中心PONS主任）
