# 东南大学

> **最后更新**: 2026年5月25日

## 学校概况

东南大学的生物医学工程为国家一级重点学科（2007年获评），在教育部学科评估中长期位列全国第一方阵。**顾忠泽教授领衔的器官芯片研究院**和**数字医学工程全国重点实验室**在类器官芯片、生物电子学方向处于国际前沿。

**官方链接**:
- 学校官网: [https://www.seu.edu.cn](https://www.seu.edu.cn)
- 研究生院: [https://seugs.seu.edu.cn](https://seugs.seu.edu.cn)
- 生物科学与医学工程学院: [https://bme.seu.edu.cn](https://bme.seu.edu.cn)

## 重点课题组与导师

### —— 器官芯片 / 类脑器官 / 生物电子学方向 ——

#### 顾忠泽 — 首席教授、博导

- **职称**: 东南大学首席教授、长江学者特聘教授、国家杰出青年基金获得者
- **职务**: 器官芯片研究院**院长**、数字医学工程全国重点实验室**主任**、生物科学与医学工程学院**院长**
- **荣誉**: 美国医学与生物工程院 Fellow、英国皇家化学会 Fellow、中国生物医学工程学会会士
- **个人主页**: [https://iirs.seu.edu.cn/2025/0604/c58381a530605/page.htm](https://iirs.seu.edu.cn/2025/0604/c58381a530605/page.htm)

**研究方向**:
1. 胶体光子晶体及其在生物医学的应用
2. 基于三维激光制造的微结构可控材料
3. **人体器官芯片的制造、测量与应用**（芯片材料、人工器官培养、支持系统）
4. **基于人工类脑器官的混合智能**（类脑器官培养、电生理信号采集与分析、智能机器人）
5. 人体器官芯片与AI融合的药物设计（器官芯片大模型、药理/毒理评价）

**学术影响力**:
- 在Nature及系列刊物上发表论文**400余篇**，被引**20000余次**
- 牵头制定并颁布**器官芯片国家标准**
- 成果入选**2023、2024年中国生命科学十大进展**
- 获国家自然科学二等奖、国家科技进步二等奖等

**2026年博士生招生**: 是（持续招收博士生及博士后）

> **关键匹配点**: 顾忠泽的"基于人工类脑器官的混合智能"方向 — 类脑器官培养 + 电生理信号采集与分析 + 智能机器人，与你的脑类器官经验 + EMG信号处理背景高度吻合！
