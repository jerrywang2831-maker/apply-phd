# 天津大学

> **最后更新**: 2026年5月25日

## 学校概况

天津大学在脑机接口领域具有独特地位：**2025年新增设置全国首个"脑机接口"一级学科博士点**，构建起从本科（2024年开设全国首个脑机接口本科专业方向）到博士的完整贯通培养体系。依托**脑机交互与人机共融海河实验室**，天大在非侵入式BCI、脑电信号处理、神经康复工程方向处于国内领先地位。

## "脑机接口"一级学科博士点

### 招生方向
- 脑机接口 + 人工智能
- 脑机接口 + 集成电路
- 脑机接口 + 临床医学
- 脑机接口 + 材料科学

### 实验室平台
依托**脑机交互与人机共融海河实验室**，拥有"临床诊疗—技术研发—产品注册—医保接入"的完整闭环链路。

### 人才需求
据明东2026年两会发言，到2030年我国脑机接口人才缺口将达20余万人。天大正加快布局脑机接口人才长周期贯通培养体系。

## 针对你的申请建议

1. **天大是国内BCI方向最具特色的高校之一**：全国首个BCI博士点，学科专注度高
2. **片上脑机接口方向与你的类脑器官经验高度吻合**：明东2024年提出的离体BCI方向，直接用干细胞培养的神经组织作为BCI接口。你的脑类器官培养+钙成像经验是独特优势
3. **"脑语者"芯片方向**：你的电路设计经验可与芯片应用层衔接
4. **非侵入式BCI康复方向**：你的EMG反馈系统经验可作为切入点
5. **建议在联系邮件中突出**：类脑器官BCI经验（这是明东团队最新方向）+ 硬件开发全栈能力
