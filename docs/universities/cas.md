# 中国科学院

> **最后更新**: 2026年5月25日

## 概况

中国科学院在神经科学和脑机接口领域拥有多个世界级研究所，其中**自动化研究所**（北京）以BCI和智能系统见长，**脑科学与智能技术卓越创新中心/神经科学研究所**（上海）以基础神经科学见长。

**关键区别**：中科院系统的博士培养与高校有所不同——更接近"师徒制"导师负责制，科研氛围更偏重项目产出，且各所之间独立招生。

## 三、申请中科院的注意事项

### 与高校申请的不同

| 方面 | 高校（如清华/北大） | 中科院 |
|------|---------------------|--------|
| 招生单位 | 学校统一招生 | 各研究所独立招生 |
| 导师权力 | 导师在委员会中有发言权 | 导师决定权更大 |
| 科研氛围 | 偏学术+教学 | 偏科研项目产出 |
| 培养模式 | 有课程学习阶段 | 更多时间在实验室 |
| 生活待遇 | 学校奖学金+导师津贴 | 研究所统一待遇（通常较高） |
| 学位授予 | 学校授予 | 中国科学院大学授予 |

### 申请建议

1. **提前联系导师是硬性要求**（自动化所明确写明）
2. **各所独立报名**：可以同时申请自动化所和神经所，互不影响
3. **竞争激烈但待遇好**：中科院博士生月收入通常高于高校
4. **住宿条件**：自动化所（北京）和神经所（上海）通常提供所内宿舍

---

## 针对你的申请建议

1. **侯增广课题组高度匹配**：BCI+康复机器人+生物医学工程背景直接对口，你的EMG反馈系统和神经技术背景可以作为切入点
2. **自动化所报名前必须联系导师**：建议2025年7-8月发送联系邮件
3. **神经所（上海）适合偏基础神经科学方向**：如果你倾向于神经发育/类器官方向，神经所的基础研究平台更合适
4. **同时申请多所不冲突**：中科院系统内各所独立招生，建议同时申请自动化所（BCI应用）和神经所（基础神经科学）以扩大机会
