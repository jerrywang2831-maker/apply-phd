# 华中科技大学

> **最后更新**: 2026年5月25日

## 学校概况

华中科技大学生物医学工程为国家一级重点学科，在**超声断层成像（UCT）**和**医学图像AI分析**方向国内领先。丁明跃团队研发的乳腺超声CT系统获国内首个超声断层成像医疗器械注册证。

**官方链接**:
- 学校官网: [https://www.hust.edu.cn](https://www.hust.edu.cn)
- 研究生院: [http://gs.hust.edu.cn](http://gs.hust.edu.cn)
- 生命科学与技术学院: [http://life.hust.edu.cn](http://life.hust.edu.cn)

#### 丁明跃 — 教授、博导

- **学院**: 生命科学与技术学院（曾任系主任、院长助理）
- **个人主页**: [http://faculty.hust.edu.cn/MINGYUEDING/](http://faculty.hust.edu.cn/MINGYUEDING/)
- **荣誉**: IEEE高级会员、湖北省生物医学工程学会副会长

**研究方向**:
- 医学图像处理与超声影像
- 颈动脉三维超声成像
- **超声断层成像**（乳腺癌早期诊断与筛查）
- 肌骨成像

**核心成果**:
- 🏆 **2025年**团队自主研发的"乳腺超声计算机断层成像系统"获**国内首个超声断层成像医疗器械注册证**
- 全球首台肌骨超声断层成像系统于2023年完成临床研究
- 获国内外专利40余项，专利转化金额1300万元
- 湖北省科技进步二等奖、华中科技大学教学名师

## 针对你的申请建议

1. **超声方向高度匹配**：UCL医用物理本科 + HIFU超声系统RA经验 + 超声成像课程基础 — 叙事线完整
2. **丁明跃/尉迟明团队**：超声断层成像前沿，工程化程度高，适合你的BME工程背景
3. **邱武团队**：AI+医学影像，你的Python/MATLAB编程能力直接可用
4. **硬件+算法双背景是优势**：大多数申请人只有单方面背景
